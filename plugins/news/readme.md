# News 插件

## 📰 功能简介

News 插件提供每日新闻和图卦推送功能，支持通过关键词触发或能力调用的方式获取最新资讯。

## ✨ 主要特性

- **今日新闻**：获取每日 60 秒读懂世界新闻图片
- **今日图卦**：获取煎蛋网每日图卦（前一天的图卦，共 2 张图片）
- **关键词触发**：支持通过特定关键词自动响应
- **能力调用**：支持通过能力接口主动调用

## 🚀 使用方法

### 关键词触发

在聊天中发送以下关键词即可触发相应功能：

| 关键词 | 功能说明 |
|--------|---------|
| `今日新闻` | 获取当天的新闻图片（60秒读懂世界） |
| `今日图卦` | 获取昨天的煎蛋图卦（2张图片） |

**示例：**
```
用户: 今日新闻
机器人: [返回今日新闻图片]

用户: 今日图卦
机器人: [返回昨日图卦图片 1/2]
机器人: [返回昨日图卦图片 2/2]
```

### 能力调用

通过插件能力接口调用：

#### news.today - 获取今日新闻

```json
{
  "capability": "news.today",
  "args": {
    "receiver": "联系人ID"
  }
}
```

**参数说明：**
- `receiver`（必填）：接收消息的联系人 ID

**返回值：**
- 成功：返回 `"none"` 并向指定联系人发送新闻图片
- 失败：返回错误信息

#### news.diagram - 获取今日图卦

```json
{
  "capability": "news.diagram",
  "args": {
    "receiver": "联系人ID"
  }
}
```

**参数说明：**
- `receiver`（必填）：接收消息的联系人 ID

**返回值：**
- 成功：返回 `"none"` 并向指定联系人发送 2 张图卦图片
- 失败：返回错误信息

## 🔧 配置说明

### 数据源

1. **今日新闻**
   - 数据源：60 秒读懂世界
   - API 地址：`https://cdn.jsdmirror.com/gh/vikiboss/60s-static-host@main/static/images/{date}.png`
   - 格式：PNG 图片
   - 更新频率：每日更新

2. **今日图卦**
   - 数据源：煎蛋网
   - API 地址：`https://penti.5aihj.com/{month}/tugua/{date}{index}.jpg`
   - 格式：JPG 图片（共 2 张）
   - 更新频率：每日更新
   - 注意：获取的是前一天的图卦

### 网络要求

插件需要访问以下外部域名：
- `cdn.jsdmirror.com` - 今日新闻图片源
- `penti.5aihj.com` - 今日图卦图片源

请确保网络环境可以正常访问这些域名。

## ⚠️ 注意事项

1. **时间差异**：今日图卦获取的是前一天的内容，因为图卦通常在次日发布
2. **发送间隔**：图卦发送时两张图片之间有 1 秒的间隔，避免消息过快
3. **网络依赖**：插件依赖外部 API，请确保网络连接正常
4. **图片格式**：
   - 今日新闻为 PNG 格式
   - 今日图卦为 JPG 格式
5. **错误处理**：如果外部 API 不可用，插件会返回相应的错误信息

## 🔍 故障排查

### 常见问题

1. **无法获取新闻图片**
   - 检查网络连接是否正常
   - 确认 `cdn.jsdmirror.com` 域名可访问
   - 检查当天的图片是否已发布

2. **无法获取图卦**
   - 检查网络连接是否正常
   - 确认 `penti.5aihj.com` 域名可访问
   - 注意图卦是前一天的内容

3. **receiver 不可为空错误**
   - 检查能力调用时是否正确传递了 `receiver` 参数
   - 确认联系人 ID 是否有效
