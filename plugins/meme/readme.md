# Meme 表情包生成插件

## 📝 插件简介

Meme 插件是一个强大的表情包生成工具，支持通过关键词快速生成各种流行的表情包。插件自动处理用户头像，支持添加自定义文字，并智能压缩图片大小以适配聊天场景。

## ✨ 主要功能

- 🎨 **丰富的表情库**：支持数百种流行表情包模板
- 👤 **智能头像获取**：自动获取用户头像生成个性化表情
- ✍️ **自定义文字**：支持在表情包上添加自定义文字内容
- 🖼️ **双图表情**：部分表情支持同时使用两个用户的头像
- 🗜️ **自动压缩**：超过 500KB 的图片自动压缩，确保发送流畅
- 🔄 **实时缓存**：支持动态刷新表情列表，无需重启

## 🚀 快速开始

### 基本使用

```
meme 揍
emoji 拍
表情 吃
```

### 查看可用表情列表

```
meme list
emoji list
表情 list
```

### 刷新表情缓存

```
meme reload
```

## 📖 使用示例

### 1. 单图表情

使用自己的头像生成表情：

```
meme 揍
```

使用他人头像生成表情（通过 @）：

```
meme 揍 @小明
```

### 2. 双图表情

部分表情需要两个用户的头像（自动使用你和被 @ 用户的头像）：

```
meme 爬 @小明
```

### 3. 带文字的表情

某些表情支持或需要添加文字参数：

```
meme 打工人 996 太累了
meme 需要补充文字的表情 第一行文字 第二行文字
```

## ⚙️ 配置说明

插件需要配置 Meme API 服务地址。在配置文件中添加：

```toml
[plugins.meme]
url = "http://localhost:2233"  # Meme API 服务地址
```

### Meme API 服务

本插件依赖 [meme-api](https://github.com/MeetWq/meme-api) 服务。你需要：

1. 部署 meme-api 服务
2. 配置服务地址到插件配置中

推荐使用 Docker 部署：

```bash
docker run -d \
  --name meme-api \
  -p 2233:2233 \
  meetwq/meme-api:latest
```

## 🎯 命令说明

### 触发前缀

插件支持以下任一前缀触发：

- `meme`
- `emoji`
- `表情`

### 特殊命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `list` | 显示所有可用的表情列表 | `meme list` |
| `reload` | 重新加载表情缓存 | `meme reload` |

### 表情参数

不同的表情有不同的参数要求：

- **单图表情**：只需要一张头像
- **双图表情**：需要两张头像（需要 @ 或引用用户）
- **文字表情**：需要提供指定数量的文字参数
- **默认文字**：部分表情有默认文字，可选择性覆盖

## 🔧 技术实现

### 核心组件

| 文件 | 功能 |
|------|------|
| `main.go` | 插件主逻辑，事件处理和流程控制 |
| `api.go` | Meme API 交互（上传、生成、下载） |
| `cache.go` | 表情信息缓存管理 |
| `types.go` | 数据结构定义 |
| `compress.go` | 图片压缩处理 |
| `message.go` | 消息解析和用户信息提取 |
| `user.go` | 用户信息收集 |

### 工作流程

```
1. 接收消息事件
   ↓
2. 匹配触发前缀（meme/emoji/表情）
   ↓
3. 解析命令（list/reload/表情关键词）
   ↓
4. 验证表情参数（图片数量、文字数量）
   ↓
5. 收集用户头像
   ↓
6. 调用 Meme API 生成表情
   ↓
7. 下载并压缩图片（如需要）
   ↓
8. 发送表情消息
```

### 图片压缩

- 阈值：500KB
- 策略：保持图片质量的同时降低文件大小
- 格式：支持 JPEG、PNG、GIF、WebP

## 📋 依赖说明

```go
require github.com/sbgayhub/golem/sdk v0.1.1
```

插件使用 Golem SDK 提供的以下能力：

- **message.Ability**：消息发送和处理
- **contact.Ability**：联系人信息获取
- **plugin.ConfigAbility**：配置管理

## 🔍 事件订阅

插件订阅以下消息类型：

- `message.TypeText`：文本消息
- `message.TypeAppQuote`：引用消息（用于获取被引用用户的头像）

## 💡 使用技巧

1. **快速生成**：记住常用表情的关键词，提高生成效率
2. **组合使用**：通过 @ 和文字参数组合，创造更有趣的表情
3. **批量查看**：使用 `list` 命令浏览所有可用表情
4. **及时更新**：Meme API 服务更新后，使用 `reload` 刷新缓存

## ⚠️ 注意事项

1. 需要先部署 Meme API 服务才能使用
2. 双图表情必须 @ 或引用其他用户
3. 部分表情对文字参数有严格要求
4. 大图片会自动压缩，可能略微影响清晰度
5. 网络不稳定可能导致生成失败，可重试

cd plugins/meme

将编译好的 `meme.plugin` 文件放置到 Golem 的插件目录中，并配置好 Meme API 服务地址。

## 🐛 故障排查

### 常见问题

**Q: 提示"未找到表情"？**  
A: 检查关键词是否正确，使用 `meme list` 查看完整列表。

**Q: 生成失败？**  
A: 检查 Meme API 服务是否正常运行，网络连接是否正常。

**Q: 图片发送失败？**  
A: 可能是图片过大或网络问题，插件会自动压缩，可重试。

**Q: 双图表情提示"需要两张图片"？**  
A: 需要 @ 一个用户或引用一条消息。
